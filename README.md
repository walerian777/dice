# dice
CLI for simulating dice rolls.

## Usage
```
Usage: dice [options]
    -f, --faces N                    Roll the dice with N faces. Default: 6.
    -c, --count COUNT                Roll the dice COUNT times. Default: 1.
    -h, --help                       Show the help message.
```

## Examples
```
$ dice
6
```

```
$ dice -f 8 -c 4
6
6
5
3
```
